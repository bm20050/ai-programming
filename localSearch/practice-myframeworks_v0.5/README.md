## AI프로그래밍 정리
- 언덕 등반 알고리즘(Min, Max) => 지역 검색
  - 전역 검색이 결과가 월등히 좋음
  - 지역 최적화는 보장
  - 알고리즘에 '제약'이 필요
- TSP, N-Queen => 유전 알고리즘
  - 전역 알고리즘 => 브루트포스
- 경사 하강법
  - 영상처리, 음성처리, 딥러닝, 시계열
  - '기울기', '절편'
